
# Flight Fare Prediction (Advanced Internship Project)

## Features
- Data Preprocessing
- Feature Engineering
- Random Forest Model
- Model Saving (.pkl)
- Streamlit Web App Deployment

## Run Locally

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## Project Structure
- data/ : Dataset
- models/ : Saved ML model
- app.py : Streamlit application
